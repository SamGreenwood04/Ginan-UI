# Ginan - UI

A simple user interface for Ginan using pyside6.

## Installation 
Check the releases tab on the right-hand side for a binary.

Run the software from source via the following. Tested with python 3.9+
```
git clone https://github.com/u7327620/Ginan-UI
cd Ginan-UI
pip install -r requirements.txt
python main.py
```
